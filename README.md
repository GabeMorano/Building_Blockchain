# Building Blockchain

This project implements a blockchain-based ledger system. The system is designed with a user-friendly web interface powered by Streamlit, allowing partner banks to securely conduct and verify financial transactions.

## Features

- **Record Data Class**: Implements a data class to encapsulate transaction data, including sender, receiver, and transaction amount.
- **Blockchain Functionality**: Enhances the existing Block data class to include transaction records, enabling the blockchain to store financial transactions.
- **User Interface**: Integrates user inputs through Streamlit to allow live interaction with the blockchain, enabling users to add and verify transactions.
- **Blockchain Integrity**: Includes functionality to validate the blockchain, ensuring the integrity and non-repudiation of the data.
